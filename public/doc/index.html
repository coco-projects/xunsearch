文档资料

Xunsearch 包含了后端服务、前端开发包（即 SDK）两大部分，在开发过程中几乎不需要关注后端， 所以我们提供的文档大部分都是 SDK 文档，用于二次开发（下载 CHM 格式，WIN 上请先右键解锁）。

为便于新手用户快速接触和了解 xunsearch 我们还制作了一份 PDF 格式的快速参考文档 《XUNSEARCH 10分钟入门》。

1. 开始&安装?

安装说明 最适合第一次接触 xunsearch 的您，以便让您先成功安装一份完整的 xunsearch。

2. 权威指南?

PHP-SDK 权威指南 是学习和入门的必经之路，它全面介绍相对应的 SDK，是学习和掌握 xunsearch 的权威资料。

最新功能和变动 是为老用户提供的了解最新功能和特点专用渠道。

通过 composer 使用 PHP-SDK 尤其是对于 yii2 用户，我提供了 AR 方式来访问索引。

3. API 文档?

PHP-SDK API 文档 是由程序自动从代码中提取并生成的 API 文档，在您读完权威指南并着手开始自己的搜索引擎时， 从这儿读取到详细的 API 用法及说明，包含了每个类对象的详细属性、方法介绍。

4. Wiki 技巧、经验?

xunsearch 使用技巧 是由用户和我们共同编写，针对一些常见的需求进行讲解和分析。 如果您在为实现某个功能疑惑不解，请尝试在这里找到您需要的答案。

5. 其他资源?

如果您还有任何其它问题或建议、需求，请到我们的论坛发帖或加入我们的 QQ 交流群 14413875，我们或热心网友都会作出解答。
